# 前言

使用egg做后台框架,用户vue做前台框架编写一个简单的博客,可以使用到简单的egg框架内容,以及vue的内容.
	
# 技术栈

vue2 + vuex + vue-router + vue-resource + webpack + ES6 + egg + mysql
